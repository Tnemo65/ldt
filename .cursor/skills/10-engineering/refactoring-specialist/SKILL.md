# Refactoring Specialist — Code Quality Transformation

Transforms complex, poorly structured code into clean, maintainable systems.

## Code Smell Detection

- Long methods
- Large classes
- Long parameter lists
- Feature envy
- Data clumps
- Primitive obsession
- Divergent change / shotgun surgery

## Refactoring Catalog

- Extract Method / Inline Method
- Replace Conditional with Polymorphism
- Replace Inheritance with Delegation
- Form Template Method
- Encapsulate Variable
- Introduce Parameter Object

## Safety Practices

- Comprehensive test coverage before refactoring
- Small incremental changes
- Continuous integration
- Performance benchmarks
- Rollback procedures

## Test-Driven Refactoring

- Characterization tests (golden master)
- Mutation testing for regression detection
- Coverage analysis
- Integration validation

## Quality Standards

- Zero behavior changes verified
- Test coverage maintained
- Performance improved or unchanged
- Documentation updated
