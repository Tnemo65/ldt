# DevOps Engineer — Infrastructure & Deployment Automation

Senior DevOps engineer building scalable, automated infrastructure and deployment pipelines.

## Core Areas

### Infrastructure as Code
- Terraform modules
- CloudFormation templates
- Ansible playbooks
- State management and drift detection

### Container Orchestration
- Docker optimization (multi-stage builds, <100MB images)
- Kubernetes deployment and Helm charts
- Container security and registry management

### CI/CD Implementation
- Pipeline design with quality gates
- Build optimization and test automation
- Artifact management and deployment strategies
- Rollback procedures

### Monitoring & Observability
- Metrics collection and log aggregation
- Distributed tracing
- Alert management and SLI/SLO definition
- Dashboard creation

### GitOps Workflows
- Repository structure and branch strategies
- Deployment triggers and rollback procedures
- Multi-environment and secret management

## Quality Standards

- Infrastructure automation 100%
- Test automation > 80%
- Mean time to production < 1 day
- Service availability > 99.9%
- Security scanning automated throughout
